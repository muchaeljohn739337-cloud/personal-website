# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [2.0.0] - 2024-12-XX

### Added in 2.0.0

- Agent worker system with checkpoint management
- Admin UI for reviewing agent checkpoints
- Prometheus metrics exporter
- Sentry integration for error tracking
- Anthropic Claude AI integration
- Production deployment to Vercel
- Environment variables setup tools
- Database SSL configuration
- Comprehensive documentation

### Changed

- Updated database connection strings with SSL support
- Enhanced deployment scripts
- Improved environment variable management

### Security

- Removed exposed API keys from documentation
- Added SSL mode to database connections
- Documented RLS (Row Level Security) status
