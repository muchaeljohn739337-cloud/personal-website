# ================================
# HOW TO GET CREDENTIALS
# ================================
# 
# RENDER:
# 1. Go to https://dashboard.render.com
# 2. Select your service → Settings
# 3. Copy "Service ID" (starts with srv-)
# 4. Go to Account Settings → API Keys
# 5. Create new API key
# 
# CLOUDFLARE:
# 1. Go to https://dash.cloudflare.com
# 2. Select your domain
# 3. Copy "Zone ID" from Overview page (right sidebar)
# 4. Go to My Profile → API Tokens
# 5. Create Token → Edit zone DNS template → Add Zone.Cache Purge permission
# 6. Copy the generated token
